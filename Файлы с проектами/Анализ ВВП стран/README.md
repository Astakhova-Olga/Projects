## Анализ динамики и корреляции ВВП стран с другими покателями
#### Краткое описание проекта
Анализ ВВП стран и его связи с другими социально-экономическими показателями за период с 1960 по 2024 годы. Разработан автоматизированный pipeline для сбора, обработки и анализа данных Всемирного банка (ключевые показатели получены через API), данные загружены в облачную СУБД PostgreSQL, создана аналитическая панель с визуализацией ключевых метрик. Сформулированы выводы о наличии корреляции между уровнем экономического развития стран и другими показателями.
#### Стек
- Python (pandas, numpy, seaborn, matplotlib, phik_matrix, requests, json, sqlalchemy, plotly.express), 
- DataLens, 
- Jupyter Notebook
