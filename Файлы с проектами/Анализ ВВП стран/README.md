# Анализ экономического развития стран и его связи с другими показателями

## Цель проекта:
Проанализировать на основе данных по всем странам, взятых на сайте Всемирного банка, за период с 1960 по 2024 гг., есть ли взаимосвязь уровня ВВП на душу населения и ряда социально-экономических показателей: прямые иностранные инвестиции, инфляция (дефлятор ВВП, годовой % роста), валовая норму охвата начальным образованием, рабочая сила с высшим образованием, уровень безработицы (общий), рождаемость (на 1000 человек), доля населения, живущего за чертой бедности, общий внешний долг.

## Стек технологий:
- Python
- Основные библиотеки: requests, pandas, sqlalchemy, seaborn, phik, matplotlib
- Datalens
- Supabase
- Jupyter Notebook

## Аллгоритм выполнения проекта
- Изучение API Всемирного банка
- Загрузка необходимых данных о странах, индикаторах и их значений с сайта Всемирного банка
- Очистка и предобработка данных в Jupyter Notebook 
- Создание базы данных
- Анализ данных и визуализация трендов (Файлы с проектами/Анализ ВВП стран/Анализ динамики и корреляции ВВП стран с другими покателями.ipynb)
- Создание аналитической панели в Datalens (https://datalens.yandex/rbbjocdik6lab)
